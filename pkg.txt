Transplication: high to high level convert
Broswers can not directly execute TypeScript code, so it needs to be converted into JavaScript. The process is called Transplication.

Key Reasons:
  JavaScript Compatibility
  Cross-Platform Execution 
  Leverging Js libraries
  Backend Compatibility
  Type Safety without Runtime Charges
